# NetFRAME Home Lab — Security Assessment (redacted public edition)

A redacted security assessment of my own home lab (NetFRAME), performed under my
own authorization as a defensive, non-destructive exercise. This public edition
has all internal IP addresses, hostnames, MAC addresses, topology specifics, and
secrets **generalized or removed**; hosts appear as generic roles (e.g. `NODE-A`,
`STORAGE-HOST`, `WORKSTATION`).

It is shared as a demonstration of security-assessment methodology and reporting:
scope and rules of engagement, evidence-driven findings ranked by severity, attack
chains, honest coverage gaps, confirmed positive controls, and a prioritized
remediation roadmap.

**[NetFRAME-Security-Assessment.pdf](NetFRAME-Security-Assessment.pdf)**

The full (unredacted) edition is kept in a separate private repository.
